# Attrition Desktop Releases

This repository hosts **binary releases only** for the Attrition Desktop game client.

## For Players

Download the game launcher from our website: [attritiongame.dev](https://attritiongame.dev)

The launcher will automatically download and install game updates from this repository.

## For Developers

**This repository contains NO source code.**

- **Source Code Repository**: [attrition-game](https://github.com/BrianSMitchell/attrition-game) (private)
- **Release Process**: Releases are created via GitHub CLI using `scripts/release.ps1`

## Releases

All game installers are published as GitHub Releases. The launcher application automatically checks for and downloads updates from this repository.

---

**Note**: This is a releases-only repository. All development happens in the private source code repository.
