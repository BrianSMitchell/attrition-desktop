# Assets Directory

This directory contains game assets and resources.

## Structure

```
assets/
├── images/          # Game images, icons, sprites
├── audio/           # Sound effects, music
├── fonts/           # Custom fonts
├── data/            # Game data files (JSON, CSV)
├── shaders/         # Graphics shaders
└── models/          # 3D models (if applicable)
```

## Guidelines

- Use web-compatible formats (PNG, JPG, WebP for images)
- Optimize file sizes for desktop distribution
- Include source files in a separate `sources/` subdirectory
- Use consistent naming conventions
- Add appropriate licenses for third-party assets
